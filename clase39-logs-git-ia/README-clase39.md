# Clase 39 - Logs, Git e IA responsable

## Objetivo
Comprender el uso de logs para diagnóstico, Git para trazabilidad e IA como apoyo responsable.

## Archivos de práctica
- logs-demo.js
- telegram-log-demo.js
- bitacora-logs-git-ia-clase39.txt
- checklist-ia-segura-trazabilidad-clase39.txt

## Aprendizajes
- Un log ayuda a diagnosticar.
- Git ayuda a dejar trazabilidad.
- La IA puede apoyar, pero no debe recibir secretos.
- Los tokens, claves y datos personales deben protegerse.
